# jibun-jiten Phase D1 指示書（署名要素「定着」の演出）

対象: Claude Code / 推奨モデル: Fable 5（このアプリの"すごい"の核。演出品質が本体。妥協不可）
前提: 台帳・決定事項リスト・jibun-jiten-デザイン言語.md を読むこと。既存実装は v0.6。

---

## これは何のフェーズか（最重要の背景）

当初からの神田さんの旗＝「無料でも有料級・親子ともに"すごい"と感じる・独自性」。
その"すごい"を一点に集約したのが署名要素「定着（てい）」。
デザイン言語で世界観（冷たい白／濃紺インク／新緑一点／道場）は確定済み。本フェーズはその核だけを実装し、実機で神田さんと詰める。
ここが気持ちよく決まれば残りの画面は同じ世界観で展開できる。だからここに時間をかける。**凡庸なアニメにしたら失敗**。

## 「定着」の定義（凡庸化を防ぐための厳密仕様）

「ポンと○が出る」のとは根本的に違う。**線そのものが2段階で変質する**演出。

- 書いている最中の線＝**生乾き**（薄い・やや灰みの青。デザイン言語 ink をそのまま使わず、薄めた状態）
- そらがきで最終画を書き終え matcher が ○ を返した瞬間、その字の全ストロークが:
  1. **沈む（settle）**: 生乾きの薄い線が、ふっと濃紺 ink(#1B2A4A) に沈み込む（色が乗る）。同時にごくわずかに線幅が締まる
  2. **にじむ（bleed）**: 定着の一瞬、線の縁がほんの少しだけ紙ににじむ（インクが繊維に広がる気配）。過剰なぼかしにしない。「気配」の量
- 所要 約0.4秒。イージングは後半をゆっくり（ease-out 系）＝「すっと吸い込まれて、止まる」
- **音なし・星なし・紙吹雪なし・キャラなし**。新緑は使ってよいが「点」まで（例: 定着後に隅へ小さな新緑の印がにじむ程度）。主役は線の変質そのもの
- 定着後の字が、そのままじぶん字典に飾られる完成形（＝保存される best/first/latest の見た目と一致させる）

## 技術要件

- 既存の描画は Phase 0 由来の canvas（desynchronized・生描画）。**書き味は絶対に落とさない**。定着演出は「書き終えた後」に走るので、書いている最中の入力遅延に一切影響させないこと
- 実装方式は Code の裁量だが、候補として: 書き終えたストロークを SVG/Canvas で再描画し、CSS transition もしくは requestAnimationFrame で色・線幅・にじみを補間
  - ⚠️台帳の教訓: 表示ロジックを rAF 依存にしすぎない（iOS 低電力時に抑制される）。rAF を使うなら低電力での劣化を確認、または CSS transition 主体に
- にじみ表現: SVG feGaussianBlur は重い/機種差が出やすい。軽量な方法を優先（線幅と不透明度の二重描画、細い外郭など）。60fps を割らないこと。iPad 実機で確認
- パラメータ（沈む時間・にじみ量・生乾きの色・締まる線幅・イージング）は**1箇所に定数化**し、実機で調整できるように。この演出は必ず往復調整が入る前提

## タスク分割

1. 単体お試しページ `settle-test.html`: 1字（例「石」）を書く→そらがき合格→定着、だけを何度も試せる最小ページ。パラメータのスライダー付き（沈む時間・にじみ量・生乾き色・イージング）。このフェーズの調整はここで回す
2. 定数を詰めたら、れんしゅうの③そらがき合格時に本組み込み（既存フローへ）
3. じぶん字典の字表示を「定着後の完成形」と一致させる（見た目の連続性）
4. バージョン表示更新 → 完了で停止・報告

## 検証（神田さんの作業。手順提示して停止）

まず settle-test.html で:

1. 実機（iPad と スマホ・通常ウィンドウ）で1字を何度も書いて定着を見る
2. 問うのはただ一つ: **「作品になった」と感じるか。気持ちいいか。**
3. 気持ちよくない場合の調整語彙 → 「もっとゆっくり／速く」「にじみ強い／弱い」「沈みが唐突／もっとぬるっと」「生乾きが濃すぎ／薄すぎ」。スライダー値をそのまま報告
4. 書き味（書いている最中の遅延）が落ちていないかも確認

## やらないこと

他画面のデザイン被せ（定着が決まってから別フェーズ）/ 音 / キャラ / 紙吹雪・星 / matcher変更 / 9級以降 / 派手さの追加（この演出は「静けさ」で勝つ）

## 完了後の展開（参考・このフェーズでは着手しない）

定着が承認されたら → じぶん字典の作り込み → れんしゅう各画面 → ためす・結果、の順にデザイン言語を被せる。核から外へ。
