# じぶん字典 デザイン言語（確定・2026-07-18）

ホームモックで神田さんが方向を承認。**全画面はこれに従う。** これがUIの正本。

## 世界観（一言）

「にぎやかなゲーム」ではなく「**自分だけの静かな道場／作品集**」。
キャラクターなし・ゲーム的演出なし。主役は常に**子ども自身が書いた字**。装飾はそれを引き立てる脇役に徹する。

## 却下した方向（戻らないための記録）

- キャラクター・マスコット → なし（独自性が死ぬ・本人が明確に不要と表明）
- ゲーム的な賑やかさ（別途開発中のプログラミングゲーム的なノリ）→ 求めていない
- 和風・習字・生成り紙に墨色 → **却下**。既存漢字アプリが全部やっており独自性にならない
- クリーム背景(#F4F1EA)＋セリフ＋テラコッタ(#D97757) → **意図的に回避**。AI生成デザインの定番＝「AIが作った印」になる

## カラートークン

| 名前 | hex | 役割 |
|---|---|---|
| paper | #F7F8FA | 背景。生成りでなく「かすかに青い冷たい白＝静かな机」 |
| paper-edge | #EDEFF3 | カードのふち・かすかな段差 |
| ink | #1B2A4A | **主役の字（子どもの筆跡）**。黒でなく万年筆の青黒 |
| ink-soft | #3A4B6E | インクの弱め（ラベル等） |
| ghost | #C7CCD4 | **手本(KanjiVG)・未習得**の脇役グレー。主役を邪魔しない |
| ghost-faint | #DDE1E7 | さらに淡い |
| grow | #3A9D6B | **達成・育つの新緑の一点**。金や赤でなく「育つ」の象徴。うるさくない |
| grow-bg | #E8F3ED | 新緑の淡い地 |

- テラコッタ・クリームは使わない（AI臭回避の肝）
- 色は増やさない。この8色で全画面。達成は新緑一点に集約

## 書体

- UI文言（ひらがな）: 丸すぎず硬すぎない知的なゴシック（Hiragino Sans系）。**まるもじに逃げない**（安っぽくなる）
- 数字（点数・画数）: 性格のある明朝系。「150」が作品のように見える扱い
- 漢字: 全部 KanjiVG（ベクター）→ 書体問題から自由。ここは既に勝っている
- 太さは2種（400/600）まで。重くしない

## 署名要素（このアプリが記憶される、たった一つ）

**「定着（てい）」の演出。** そらがきで1字を書き上げた瞬間、線が濃紺のインクとして紙に染み込んで定着する一瞬（約0.4秒）。ファンファーレ・星・キャラの反応は出さない。「自分の線が作品になった」手触りだけを残す。

- 理由: 賑やかな演出は3日で飽きる。「自分の線が美しく定着する気持ちよさ」は飽きない。親が"すごい"と言うのは点数でもキャラでもなく、わが子の手の字がきれいに飾られた瞬間
- じぶん字典に並ぶのは、この定着した一字一字
- 大胆さはここ一箇所に集中。他画面は静かに保つ（frontend-designスキル「spend your boldness in one place」）

## レイアウト原則

- ホームの主役は**メニューボタンではなく「これまで書いた字の壁」**。開いた瞬間「自分の場所だ」と感じさせる
- 濃紺インクの塊は画面に1つだけ置き、そこに新緑がにじむ（＝定着の予告）
- 1画面1目的・大きなタップ領域・応答速度優先。アニメは短く、やりすぎない
- 文言は全ひらがな・禁止語（×・ざんねん・まちがい・しっぱい）継続・texts.js集約
